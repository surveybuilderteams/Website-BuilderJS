# Website-BuilderJS
A JavaScript library that allow you to create a website using plain JavaScript


***

# Code

```js
let web = new website({
tag: "{element tag}",
id: "{element id}",
classN: "{element class}",
name: "{element name}",
attr:[
    {
type: "{attr_type}",
val: "{value}"
}
],
def_text: "{text}",
parent: "{parent}",
styles: ["{css path}"],
scripts: ["{js path}"]
});
```
